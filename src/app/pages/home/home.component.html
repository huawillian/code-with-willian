Today's Coding Toy Problem

Treasure Island

You bought an ancient map of the surrounding islands and the locations of the hidden treasures on each of those islands. However, you only have enough supplies to travel to and from one of these islands. Your goals is to find the island with the most amount of treasure.

The map is a 2 dimensional matrix where each item can either be 'W', 'L', or 'T', which stands for water, landmass, and buried treasure respectively. A cluster of landmasses is considered an island if they are connected to each other by being immediately adjacent to each other or through other connected land masses. Buried treasure are also landmasses. The output should be a list of the indexes representing the island with the most connected treasures.

Example

Input 

    [[ 'L', 'L', 'O', 'O'],
      [ 'T', 'L', 'O', 'T'],
      [ 'L', 'O', 'O', 'L'],
      [ 'L', 'O', 'T', 'L']]

Output 

    [[1, 3],
    [2, 3],
    [3, 2],
    [3, 3]]

Reason

There are only 3 treasures in this island: [1, 0], [1, 3], and [3, 2]. The first treasure is not connected to any other treasures through landmasses. The second and third treasures are connected by landmasses: [2, 3] and [3, 3].
